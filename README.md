E-Commerce Sales Performance Analytics Dashboard
Project Overview
This project features a comprehensive 2-page interactive Power BI dashboard designed to analyze $158M in Sales Data. The goal was to transform raw transactional data into actionable business insights, focusing on revenue trends, customer demographics, and discount impacts.

🚀 Key Features
1. Executive Summary (Page 1)
High-Level KPIs: Tracked core business metrics including Total Sales ($158M), Total Orders (55K), and Average Order Value ($2.88K).

Optimized Interactions: Implemented Edit Interactions to ensure clean cross-filtering across Category, Gender, and Location visuals, preventing visual clutter.

Customer Segmentation: Analyzed sales distribution by Age Groups and Gender to identify the primary target audience.

Payment Distribution: Visualized revenue share across Credit Cards, UPI, and Net Banking using Donut charts.

2. Granular Drill-Through (Page 2)
Deep-Dive Navigation: Enabled Drill-through functionality to allow users to navigate from high-level categories to a detailed transaction table.

Advanced Formatting: * Data Bars: Applied to the "Net Sales" column to visually distinguish high-value transactions instantly.

Status Indicators: Used color-coded (Red/Green) formatting for "Discount Availed" (Yes/No) for better readability.

User Experience: Integrated a functional Back Button for seamless navigation between the summary and detail pages.

🛠️ Tech Stack & Skills
Tool: Microsoft Power BI Desktop

Data Transformation: Power Query (Cleaning & Data Shaping)

Concepts: DAX Measures, Data Modeling, Drill-through, Cross-filtering, Conditional Formatting.

📈 Key Business Insights
Top Performer: The Electronics category leads the market with $47M in total sales.

Top Locations: Mumbai and Delhi emerged as the highest revenue-generating regions.

Discount Impact: Identified a direct correlation between discount availability and purchase volume among younger demographics.
